# Wokku Homebrew Tap

Homebrew formulae for [Wokku](https://wokku.dev) CLI.

## Install

```bash
brew install johannesdwicahyo/tap/wokku
```

## Usage

```bash
wokku auth:login
wokku apps
wokku deploy ghost --server my-server
```

See [wokku.dev/docs/cli/commands](https://wokku.dev/docs/cli/commands) for the full command reference.
